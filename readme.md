# SmileSion的个人博客

本博客特点如下：

- 本博客基于hugo，使用papermod主题，在此基础上大幅度修改部分内容，保持个人风格
- 添加了留言板、瀑布照片墙、当下页面、每日一言等功能
- 碎碎念类型文章定制模板，排版高级舒适、具备回到顶部功能
